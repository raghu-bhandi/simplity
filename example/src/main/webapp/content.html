Getting Started

To start developing web applications with Simplity, please follow the instructions below:

  • Create a new project
    • Create a new "Dynamic Web Project" in eclipse. Use jre 1.6 or above.
    • Using your favorite build process, ensure that simplity-core.jar is accessible. (Simplest thing to do is to put that in WEB-INF/lib folder, and use that as external jar on eclipse build path)
    • Configure web.xml to have the following entry.
      <servlet>
      <servlet-name>s</servlet-name>
      <servlet-class>org.simplity.http.Serve</servlet-class>
      </servlet>
      <servlet-mapping>
      <servlet-name>s</servlet-name>
      <url-pattern>*._s</url-pattern>
      </servlet-mapping>
    • Copy comp and schema folders of Simplity to WEB-INF folder.
      • Comp is where we will be adding components of our applications.
      • Schema has the grammar that will guide us in creating the components.
      • Add the two .xsd files as reference using
        windows -> preference -> xml -> xml-catalog -> user specified entries
    • Copy simplity folder under WebContent. This has just one .js file that we are going to use on the client.
  • Creating the service
    • Create a file named helloWorld.xml under comp/service/tp with the following content.
      <?xmlversion="1.0"encoding="UTF-8"standalone="no"?>
      <service xmlns="http://www.simplity.org/schema" name="helloWorld">
      <actions>
      <setValue actionName="sayHello" fieldName="hello" fieldValue="Hello World" />
      </actions>
      <outputData fieldNames="hello" />
      </service>
  • Creating the client
    • Create helloWorld.html with the following content
      <!DOCTYPEhtml>
      <html>
      <head>
      <meta charset="ISO-8859-1">
      <title>Hello World</title>
      </head>
      <body onload="Simplity.getResponse("helloWorld").then(Simplity.pushDataToPage, Simplity.showMessage);">
      <h1 id="hello"></h1>
      <script type="text/javascript" src="sayHello.js"></script>
      <script type="text/javascript" src="js/simplity.js"></script>
      </body>
      </html>

We are done with Helloworld application
Go ahead and say Hello World by running your app under eclipse/tomcat
