Example

<?xml version="1.0" encoding="UTF-8" standalone="no"?>
<dataTypes name="myTypes" xmlns="http://www.simplity.org/schema">
<components>
<numericDataType name="id" description="Internally generated number by the system as primary key for table" maxValue="99999999999999" messageName="invalidId" minValue="0" />
<numericDataType name="number" minValue="0" description="Non-negative whole number with a max of 14 digits is accepted." maxValue="99999999999999" messageName="invalidNumber" nbrFractionDigits="0" />
</components>
</dataTypes>

Attributes

Attribute name Required? Description
minValue no minimum integer value, ignoring fractions
maxValue no maximum integral value ignoring fractions
nbrFractionDigits no 0 means this is integral. Number of decimal places to maintain. For example if you want to have amount with cents/paisa like 12.34, you would set this to 2.
